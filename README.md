# mayfly-go

#### 介绍
简单基于DDD(领域驱动设计)分层架构实现web版mysql,redis,linux统一操作管理平台

### 开发语言与主要框架
- 前端：typescript、vue3、element-plus
- 后端：golang、gin、gorm
