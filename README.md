# 🌈mayfly-go

### 介绍
简单基于DDD(领域驱动设计)分层架构实现的web版 **linux、数据库（mysql）、redis、mongo统一管理操作平台**


### 开发语言与主要框架
- 前端：typescript、vue3、element-plus
- 后端：golang、gin、gorm
